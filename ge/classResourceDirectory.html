Provides a searchable directory of game resources.

Contents

Provides a directory of game resources which mirrors a recursive directory listing of files. Entries in the directory are intended to carry some guarantee of validity. For instance, if a tilemap is found in the directory then we can assume that it at least contains some sort of valid xml, and that we may refer to it in javascript with the assumption that it will parse quickly on the c++ side.

    ResourceDirectory dir;
    dir.BuildFromPath("res");
    const ResourceReference* ref = dir.Find("res:examples:test_hexagonal_tile_60x60x30.png");
