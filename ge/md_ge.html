The shell

There are two global classes which are provided to the shell when it initializes

  • Global.Application
  • Global.Module

Global.Application

Global.Application is intended to provide an interface to the underlying SDL application.

  • Global.Application.running()
  • Global.Application.title(new_title)
  • Global.Application.rootScene()

Global.Module

Use Global.Module to install additional modules via require as follows:

module = Global.Module()
module.require("Isolate")
hs = Isolate.HeapStatistics()
hs.update()
hs.total_heap_size()

Modules available through the require mechanism:

  • Isolate
  • Object
  • Scene
  • Camera
  • Others

Isolate

module = Global.Module()
module.require("Isolate")

Use this to do various things with the v8 isolate the shell is running.

  • hs = Isolate.HeapStatistics()
    Constructs a new class with the following member functions:
    • <undefined> hs.update()
      calls v8::Isolate::GetHeapStatistics() on the current v8 isolate

    • <Number> hs.total_heap_size()
    • <Number> hs.total_heap_size_executable()
    • <Number> hs.used_heap_size()
    • <Number> hs.heap_size_limit()
      these functions return heap statistics values generated by the last call to hs.update():
  • [Others]
